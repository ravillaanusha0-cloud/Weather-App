# Weather-App

This is a simple Python program that shows current weather details of a city using an online weather API.

--------------------------------------------------

Features

- Takes city name as input
- Displays temperature in Celsius
- Displays weather condition
- Uses live weather data

--------------------------------------------------

Technologies Used

- Python
- Requests library
- OpenWeatherMap API
- JSON

--------------------------------------------------

How It Works

1. User enters a city name
2. Program sends a request to the weather API
3. API returns data in JSON format
4. Program extracts and displays weather details

--------------------------------------------------

Requirements

- Python 3
- requests library

To install requests:
pip install requests

--------------------------------------------------

How to Run

1. Open terminal / command prompt
2. Run the file:
   python weather_app.py
3. Enter the city name

--------------------------------------------------

Sample Output:

Enter the city: Tokyo                       
Temperature:3.83
Condition: broken clouds
Humidity:44%
